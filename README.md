# AVAXFUN, Meme Token Launcher on Avalanche

Permissionless meme token factory and a simple web UI to deploy ERC20 tokens on Avalanche C-Chain.

Live site: https://avaxfun.net

## What it does
- Simple Launch: deploy token, 100% supply to creator
- Launch + Liquidity: deploy token, add liquidity, 80% to pool, 20% to creator
- Fee: 0.15 AVAX per launch
- Limits: max 5 launches per address per day, max 1T supply, min 0.1 AVAX liquidity
